# Security Policy

Matter Core is experimental software. Please do not use public issues for sensitive vulnerability details.

## Reporting

Report security concerns privately to the repository owner through GitHub, or by opening a minimal private advisory when available.

Include:

- Affected crate, command, or runtime component.
- Reproduction steps.
- Expected impact.
- Suggested fix, if known.

## Scope

Security-sensitive areas include:

- File, process, network, and backend access from Matter programs.
- Package/project manifest resolution.
- Polyglot bridge execution.
- Web/visual runtime serving and event handling.
- Bytecode deserialization and VM execution.

## Supported Versions

The current `main` branch is the only supported development line.
