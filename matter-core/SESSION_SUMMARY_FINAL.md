## 🔥 FEATURES DEMONSTRADAS

### 1. Effect Inference (Zero Boilerplate)
**Todos os 8 exemplos** usam inferência automática de efeitos.

```matter
fn process(data: string) -> result {
    print "Processing...";
    db.query("INSERT INTO data VALUES (?)", data);
    net.post("https://api.com/sync", data);
    return ok;
    // Efeitos inferidos: io, db, network
}
```

### 2. Effect Handlers (Production-Ready)
**Todos os exemplos de produção** usam handlers.

```matter
with logging, tracing, retry, auth, rate_limit {
    process_order(order);
}
```

### 3. Hot Reload (10x Faster)
**Todos os exemplos** podem ser desenvolvidos com hot reload.

```bash
matter hotreload app.matter
```

### 4. Gradual Typing (Flexibility + Safety)
**Todos os exemplos** usam tipos opcionais.

```matter
fn prototype(data) { return data * 2; }
fn production(data: int) -> int { return data * 2; }
```

### 5. 3 Backends (Unique!)
**Todos os exemplos** podem ser executados de 3 formas.

```bash
matter run app.matter              # Bytecode (1x)
matter compile-llvm app.matter     # LLVM (100x)
matter compile-native app.matter   # Native (50-100x)
```
