O que e Matter Core

Um runtime de linguagem feito para ser pequeno, auditavel e controlavel.

O Matter Core nao tenta substituir Rust, Python ou JavaScript. Ele cria uma camada menor para rodar logica controlada, inspecionar estrutura do programa e conectar ferramentas externas por comandos estaveis.

01

Codigo vira bytecode

Programas .matter passam por parser, AST e bytecode antes da VM executar. Isso abre espaco para validacao, diagnostico e portabilidade.

02

Reflexao antes da execucao

Comandos como reflect-json e reflexive-guard-json ajudam ferramentas a entenderem o programa antes de confiar nele.

03

CLI para maquinas

matter capabilities-json, diagnosticos e manifests tornam o runtime simples de plugar em agentes, scripts, instaladores e testes automatizados.

Para quem

Feito para builders que precisam de execucao previsivel.

Times de IA e agentes

Criadores de DSLs

Quando voce quer uma linguagem de dominio com VM, exemplos, testes e caminho para embed.

Automacao interna

Quando scripts precisam ser distribuidos, diagnosticados e verificados por checksum antes de rodar.

Pesquisa de linguagens

Quando o objetivo e estudar parser, bytecode, VM, reflexao e tooling em um sistema real.

Demo principal

Uma regra de agente que roda, pode ser refletida e pode ser bloqueada.

O exemplo agent_policy_demo.matter calcula risco para uma acao de agente. O ponto nao e o calculo em si. O ponto e que a regra vira programa, bytecode, saida de VM e estrutura JSON inspecionavel.

script .matter bytecode VM reflect-json guard-json 
matter run examples\agent_policy_demo.matter
matter reflect-json examples\agent_policy_demo.matter
matter reflexive-guard-json examples\agent_policy_demo.matter

Comparacao honesta

Onde Matter entra.

Node/Python

Otimos para ecossistema grande. Matter entra quando voce quer regra pequena, VM propria e inspecao antes de executar.

Rust

Otimo para construir sistemas. Matter entra como camada distribuivel para usuario final sem exigir toolchain Rust.

Arquivos de config

Simples, mas limitados. Matter entra quando a configuracao precisa de logica, funcoes, eventos e guarda automatizada.

Por que baixar agora

O beta prova a base: instalar, rodar, inspecionar, diagnosticar e remover com seguranca.

Quem testar agora ajuda a validar o caminho mais importante do produto: uma linguagem experimental que pode ser baixada por qualquer pessoa, verificada sem fe cega e operada por humanos ou maquinas.

Caminho recomendado

Setup EXE agora, assinatura real depois do certificado.

.\matter-core-beta-setup.exe

# ou clique duplo no fallback CMD:
.\install-matter-beta.cmd

# ou manualmente:
powershell -ExecutionPolicy Bypass -File .\install-release-zip.ps1 `
  -ZipPath .\matter-core-windows-x64.zip `
  -ChecksumJsonPath .\release-checksums.json `
  -Sha256Path .\SHA256SUMS.txt

matter run examples\first_run.matter
matter run examples\agent_policy_demo.matter
matter capabilities-json

Status beta

Polido para teste publico, honesto sobre limites.

Este beta entrega setup EXE, instalacao local, checksum, diagnostico e desinstalacao segura. Ainda faltam assinatura de codigo real, MSI assinado, auto-update e instaladores Linux/macOS.

Roadmap

Proximos passos para sair de beta.

  1. Instalador MSI ou EXE assinado.
  2. Auto-update e verificacao de release.
  3. Mais exemplos reais para agentes, DSLs e automacao interna.
  4. Linux/macOS depois do Windows x64 estabilizar.
