feat: REPL interativo para experimentação com Matter

Sprint 4: REPL Interativo - Shell Interativo Funcional

NOVO COMANDO:
- repl                  Inicia shell interativo REPL

FUNCIONALIDADES:
- Shell interativo com prompt numerado [N]>
- Execução imediata de código Matter
- Multi-line input para blocos (funções, if, loops)
- Histórico de comandos armazenado
- Tratamento robusto de erros
- Interface visual profissional

COMANDOS ESPECIAIS:
- :help       Mostra ajuda do REPL
- :quit       Sai do REPL
- :clear      Limpa a tela
- :reset      Reinicia runtime
- :vars       Lista variáveis
- :backends   Lista backends
- :history    Mostra histórico

DETECÇÃO MULTI-LINE:
- Detecta automaticamente blocos que precisam múltiplas linhas
- Prompt muda para `...` em modo multi-linha
- Linha vazia executa bloco acumulado
- Contagem de chaves para detectar fim do bloco

CASOS DE USO:
- Experimentação rápida com a linguagem
- Teste de expressões e funções
- Prototipagem de ideias
- Debugging interativo
- Aprendizado hands-on
- Demonstrações ao vivo

LIMITAÇÕES CONHECIDAS:
- Estado não persistente entre comandos (runtime isolado)
- Sem autocomplete (futuro: Sprint 4.3)
- Sem navegação de histórico com setas (futuro: Sprint 4.2)
- Sem syntax highlighting (futuro: Sprint 4.4)

[3]> fn soma(a, b) {
