feat: Expand standard library with time, random, and json backends

## Summary

Significantly expanded Matter Core's standard library with three new backends
and additional methods for existing backends, improving developer productivity
and enabling real-world use cases.

## New Backends

### TimeBackend ⏰
- `time.now()` - Get current timestamp (ms since Unix epoch)
- `time.sleep(ms)` - Sleep for specified milliseconds

Use cases: timestamps, performance measurement, delays

### RandomBackend 🎲
- `random.int()` - Random integer (full range)
- `random.int(max)` - Random integer [0, max)
- `random.int(min, max)` - Random integer [min, max)
- `random.bool()` - Random boolean
- `random.choice(list)` - Random element from list

Use cases: games, simulations, testing, data generation

### JsonBackend 📄
- `json.stringify(value)` - Convert value to JSON string
- `json.parse(json)` - Parse JSON string to value

Use cases: API communication, data serialization, logging

## Enhanced Backends

### MathBackend
- `math.mod(a, b)` - Modulo operation
- `math.clamp(value, min, max)` - Clamp value between min and max
