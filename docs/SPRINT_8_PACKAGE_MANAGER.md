# Sprint 8: Package Manager

**Status:** 🔄 EM PLANEJAMENTO  
**Data:** Junho 2026  
**Prioridade:** 📦 ALTA

## Objetivo

Implementar package manager básico para Matter Core, permitindo compartilhamento e reutilização de código.

## Motivação

### Problema Atual (v0.3)
- ❌ Sem sistema de pacotes
- ❌ Sem resolução de dependências
- ❌ Sem versionamento
- ❌ Código duplicado entre projetos
- ❌ Difícil compartilhar bibliotecas

### Solução (v0.4)
- ✅ Package manager (`matter pkg`)
- ✅ Manifesto de pacote (`matter.toml`)
- ✅ Resolução de dependências
- ✅ Versionamento semântico
- ✅ Registry local (futuro: remoto)

## Arquitetura

### Package Structure

```
my-package/
├── matter.toml          # Manifesto do pacote
├── src/
│   ├── main.matter      # Entry point
│   ├── lib.matter       # Biblioteca
│   └── utils.matter     # Utilitários
├── tests/
│   └── test_main.matter
├── examples/
│   └── example.matter
└── README.md
```

### matter.toml Format

```toml
[package]
name = "my-package"
version = "0.1.0"
authors = ["Your Name <you@example.com>"]
description = "A Matter package"
license = "MIT"
entry = "src/main.matter"

[dependencies]
math-utils = "1.0.0"
string-helpers = { version = "0.5.0", path = "../string-helpers" }

[dev-dependencies]
test-framework = "0.2.0"

[stdlib]
path = "~/.matter/stdlib"

[store]
path = ".matter_store.json"
```

## CLI Commands

### Package Management

```bash
# Criar novo pacote
matter pkg new my-package
matter pkg new my-lib --lib

# Inicializar pacote existente
matter pkg init

# Adicionar dependência
matter pkg add math-utils
matter pkg add string-helpers --version 0.5.0
matter pkg add local-lib --path ../local-lib

# Remover dependência
matter pkg remove math-utils

# Atualizar dependências
matter pkg update
matter pkg update math-utils

# Listar dependências
matter pkg list
matter pkg tree

# Verificar pacote
matter pkg check
matter pkg verify

# Publicar pacote (futuro)
matter pkg publish
```

### Build & Run

```bash
# Build pacote
matter pkg build
matter pkg build --release

# Run pacote
matter pkg run
matter pkg run --release

# Test pacote
matter pkg test
matter pkg test --verbose

# Clean build artifacts
matter pkg clean
```

## Dependency Resolution

### Semantic Versioning

```toml
# Versão exata
math-utils = "1.0.0"

# Versão compatível (^)
math-utils = "^1.0.0"  # >= 1.0.0, < 2.0.0

# Versão minor compatível (~)
math-utils = "~1.2.0"  # >= 1.2.0, < 1.3.0

# Range
math-utils = ">= 1.0.0, < 2.0.0"

# Wildcard
math-utils = "1.*"
```

### Dependency Graph

```
my-app (0.1.0)
├── math-utils (1.0.0)
│   └── core-lib (0.5.0)
├── string-helpers (0.5.0)
│   └── core-lib (0.5.0)  # Reused
└── json-parser (2.0.0)
    └── string-helpers (0.5.0)  # Reused
```

### Lock File (matter.lock)

```toml
# This file is automatically generated by Matter Package Manager
# Do not edit manually

[[package]]
name = "math-utils"
version = "1.0.0"
source = "registry+https://packages.matter-lang.org"
checksum = "abc123..."

[[package]]
name = "core-lib"
version = "0.5.0"
source = "registry+https://packages.matter-lang.org"
checksum = "def456..."

[[package]]
name = "string-helpers"
version = "0.5.0"
source = "path+../string-helpers"
checksum = "ghi789..."
```

## Package Registry

### Local Registry

```
~/.matter/
├── registry/
│   ├── index/
│   │   ├── math-utils.json
│   │   └── string-helpers.json
│   └── packages/
│       ├── math-utils-1.0.0/
│       └── string-helpers-0.5.0/
├── cache/
│   └── downloads/
└── config.toml
```

### Registry Index Format

```json
{
  "name": "math-utils",
  "versions": [
    {
      "version": "1.0.0",
      "description": "Math utilities for Matter",
      "authors": ["Author Name"],
      "license": "MIT",
      "dependencies": {
        "core-lib": "^0.5.0"
      },
      "checksum": "abc123...",
      "published_at": "2026-05-01T00:00:00Z"
    }
  ]
}
```

## Import System

### Module Imports

```matter
# Import from dependency
import math_utils
import string_helpers

# Use imported modules
let result = math_utils.sqrt(16)
let upper = string_helpers.upper("hello")
```

### Relative Imports

```matter
# Import from same package
import "./utils"
import "../lib"

# Use
let value = utils.helper()
```

### Selective Imports (Futuro)

```matter
# Import specific functions
import { sqrt, pow } from math_utils
import { upper, lower } from string_helpers

# Use directly
let result = sqrt(16)
let text = upper("hello")
```

## Exemplos

### Exemplo 1: Criar Pacote

```bash
$ matter pkg new my-app
Created package `my-app` v0.1.0
    - src/main.matter
    - matter.toml
    - README.md

$ cd my-app
$ matter pkg run
   Compiling my-app v0.1.0
    Finished in 0.5s
     Running target/my-app.mbc
Hello, Matter!
```

### Exemplo 2: Adicionar Dependência

```bash
$ matter pkg add math-utils
    Updating registry index
   Resolving dependencies
  Downloaded math-utils v1.0.0
   Compiling math-utils v1.0.0
   Compiling my-app v0.1.0
    Finished in 1.2s
```

**matter.toml:**
```toml
[dependencies]
math-utils = "1.0.0"
```

**src/main.matter:**
```matter
import math_utils

let result = math_utils.sqrt(16)
print result  # 4
```

### Exemplo 3: Biblioteca Local

```bash
$ matter pkg new my-lib --lib
$ cd my-lib
```

**src/lib.matter:**
```matter
fn greet(name) {
    return "Hello, " + name + "!"
}

fn farewell(name) {
    return "Goodbye, " + name + "!"
}
```

**Usar em outro projeto:**
```bash
$ cd ../my-app
$ matter pkg add my-lib --path ../my-lib
```

**src/main.matter:**
```matter
import my_lib

print my_lib.greet("World")
print my_lib.farewell("World")
```
